## PicReview (https://willowding.github.io/picreview)
为图片审核设计的多人实时协作工具。
A real-time collaborative review tool for AI-generated picture book illustrations.

## 使用方法 / How to Use
打开网址，输入名字，选择角色后进入。
Open the link, enter your name, and choose your role.
| 角色 Role | 权限 Permissions |
|-----------|-----------------|
| **上传方 Uploader** | 上传 / 删除 / 替换图片 Upload / Delete / Replace images |
| **审核员 Reviewer** | 通过 / 打回 / 评论 Approve / Reject / Comment |

## 审核标准 / Review Criteria
- 无描边 No outlines on characters or scenes
- 无豆豆眼 No bead eyes — characters must have visible whites
- 图案圆润，形象可爱 Rounded shapes, cute characters
- 动物形态符合真实 Realistic animal anatomy
- 场景道具符合逻辑 Logical props and settings
- 颜色柔和，文字清晰 Soft colors, legible text
- 风格统一，适合3–6岁 Consistent style, suitable for ages 3–6
- 无敏感或侵权元素 No sensitive or copyrighted content

## 操作说明 / Tips
- 弹窗内按 ← → 键可左右切换图片，无需逐个关闭
- Use ← → arrow keys to navigate between images in the preview
- 批量上传自动检测重复，相同图片会跳过
- Duplicate images are detected automatically and skipped
